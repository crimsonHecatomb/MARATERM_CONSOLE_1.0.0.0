## Features:
### Top bar:
- Displays hour, time and date
### Left Widgets:
- Storage monitor
  - (Opens C drive when clicked)
- Upload and download speed monitor
  - (Button opens the Network and Internet settings page)
### Rigt Widget:
- System monitor
## Bottom bar:
- Clickable buttons
- Start menu
- Discord
- Default web browser
- Direct internet search
- Volume control via scroll wheel
- Audio output device control via scroll wheel
- Network status

## Installation:
1. Download [Rainmeter](https://www.rainmeter.net/)
2. Download the MARATERM_CONSOLE_1.0.0.0.rmskin file in [Releases](https://github.com/crimsonHecatomb/MARATERM_CONSOLE_1.0.0.0/releases/tag/Release)
3. Double-click the MARATERM_CONSOLE_1.0.0.0.rmskin file
4. Proceed with installation

### Alternative method:
1. Download [Rainmeter](https://www.rainmeter.net/)
2. Download the repository
3. Enable the widgets within Rainmeter
